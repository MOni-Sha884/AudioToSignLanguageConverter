Key Features

Built for Accessibility & Intelligence

🎯
Context Detection
Automatically identifies whether the conversation is in a medical, educational, legal, emergency, or casual context — adapting vocabulary and priority accordingly.
💬
Emotion Recognition
Detects emotional tone — happy, urgent, questioning, sad, angry — to guide the avatar's facial expressions and signing speed for natural communication.
📐
ISL Grammar Engine
Transforms English word order into ISL's Topic-Comment SOV structure, moving time expressions forward and question words to the end.
🎙️
Multi-Format Audio
Accepts live microphone input or uploaded audio files (MP3, WAV, OGG, FLAC, M4A). Powered by Google Speech API with offline fallback.
🧑‍💻
3D Avatar Signing
A fully animated 3D avatar renders each ISL sign using SiGML-based motion files, with smooth transitions between signs and words.
🔄
Adaptive Learning
Users can rate translations and submit corrections. Every session is logged to enable continuous model improvement over time.
